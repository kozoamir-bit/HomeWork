# כיתה א׳1 – לוח הכיתה

עמוד שמציג אוטומטית מה למדנו היום ומה שיעורי הבית, לפי הקובץ השבועי בגוגל דוקס.

## התקנה בפרויקט הקיים

העתק את הקבצים הבאים לפרויקט:

```
lib/parseDoc.ts          ← הלוגיקה של שליפת הנתונים
app/page.tsx             ← הדף הראשי
app/page.module.css      ← עיצוב הלוח
app/layout.tsx           ← עדכן/מזג עם ה-layout הקיים שלך
app/globals.css          ← הוסף לקובץ הקיים שלך
```

## אין dependencies חדשים!

הקוד משתמש רק ב-`fetch` מובנה של Next.js.  
אין צורך בהתקנת חבילות חדשות.

## איך זה עובד

1. הדף הוא **Server Component** – רץ אצל Vercel, לא בדפדפן
2. שולף את גוגל דוקס (URL ציבורי, ללא API key)
3. מנתח את טבלת ה-HTML ישירות – ללא ספריות
4. **Cache של 30 דקות** – לא פוגע בגוגל בכל רענון
5. מזהה אוטומטית את היום לפי שעון ישראל

## עדכון ה-Google Doc URL

אם תרצה להחליף את הקובץ, ערוך את השורה הראשונה ב-`lib/parseDoc.ts`:

```ts
const DOC_URL = "https://docs.google.com/document/u/0/d/YOUR_DOC_ID/mobilebasic";
```

## מבנה הטבלה הצפוי

הקוד מצפה לטבלה עם השורות הבאות (לפי סדר):
- שורה 0: כותרות ימים (ראשון-שישי עם תאריכים)
- שורה 1: קריאת בוקר
- שורות 2-4: שפה / חשבון / תורה (מה שלמדו)
- שורה 5: ריקה
- שורות 6-8: שפה / חשבון / אחר (שיעורי בית)

אם המורה תשנה את מבנה הטבלה, עדכן את מיפוי השורות ב-`parseDoc.ts`.
